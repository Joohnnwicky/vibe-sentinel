# Vibe Sentinel - 屏幕活动监控报警器

## 项目概述
监控屏幕画面变化，当检测到画面静止超过阈值时自动发出蜂鸣报警，确保不会错过需要决策的时机。

## 核心功能
- 实时监控屏幕画面像素变化
- 空闲超时检测（画面静止）
- 蜂鸣报警系统
- **可视化区域选择** - 图形界面拖动选择监控区域
- **区域预览** - 显示所选区域的实时预览图
- **多显示器支持** - 可选择监控哪个显示器
- **可配置参数** - 空闲阈值、蜂鸣频率、蜂鸣次数

## 技术栈
- Python 3.9+
- tkinter: GUI图形界面
- mss: 高性能屏幕截图
- numpy: 图像像素比较
- winsound: Windows蜂鸣

## 约束条件
- [execute] Windows平台优先
- [execute] 使用mss进行屏幕截图
- [execute] 使用tkinter构建GUI界面
- [verify] 所有功能需要可独立测试
