# CONTEXT.md - 项目上下文

## 问题背景
在进行vibe coding时，Claude Code会在需要用户决策时等待响应。用户可能因为专注于其他任务而错过了这些决策时机，导致效率降低。

## 解决方案
开发一个轻量级的屏幕活动监控脚本，当检测到用户长时间没有与电脑交互（鼠标/键盘）时，发出可自定义的蜂鸣报警提醒。

## 关键设计决策
1. **活动检测**: 使用pynput库监听全局鼠标和键盘事件
2. **报警机制**: 使用winsound.Beep生成不同频率/模式的蜂鸣
3. **配置方式**: 通过命令行参数或配置文件调整阈值
4. **运行模式**: 持续后台运行，直到用户主动停止

## 状态
项目初始化中，核心技术栈已确定。
