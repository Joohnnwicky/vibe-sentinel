# Vibe Sentinel

屏幕活动监控报警器 - 监控指定区域画面变化，当检测到空闲时自动发出警报（支持自定义MP3）。

## 功能特性

- **区域选择监控** - 支持自定义选择屏幕上的任意区域进行重点监控
- **多显示器支持** - 自动检测并支持多个显示器
- **像素变化检测** - 通过实时对比截图像素变化判断画面是否有活动
- **自定义MP3报警** - 支持选择本地MP3文件作为警报声音
- **蜂鸣报警** - 当画面静止超过设定阈值时自动发出蜂鸣声（可隐藏）
- **实时预览** - 选择区域后在界面中显示预览缩略图
- **可配置参数** - 支持调整空闲阈值、MP3播放次数、显示器编号等参数
- **滚动界面** - 支持鼠标滚轮滚动查看所有设置
- **监控状态管理** - 开始监控后自动禁用区域选择，避免误操作

## 适用场景

本工具专为 Vibe Coding 场景设计，例如在使用 Claude Code 进行 AI 辅助编程时，监控终端窗口是否在等待用户决策。当画面长时间静止时发出警报，提醒用户及时响应。

同样适用于：
- 监控下载进度
- 监控渲染/构建进度
- 任何需要等待画面变化的场景

## 安装

### 方式一：使用启动脚本（推荐）

1. 双击 `启动监控.bat` 文件
2. 脚本会自动检查依赖并启动程序

### 方式二：从源码运行

1. 确保已安装 Python 3.8+
2. 克隆仓库
   ```bash
   git clone https://github.com/Joohnnwicky/vibe-sentinel.git
   cd vibe-sentinel
   ```
3. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
4. 运行程序
   ```bash
   python vibe_sentinel_gui.py
   ```

## 使用方法

1. **启动程序** - 运行 `python vibe_sentinel_gui.py` 或 `启动监控.bat`

2. **选择监控区域**
   - 点击界面上的"选择区域"按钮
   - 在屏幕上拖动鼠标框选需要监控的区域
   - 松开鼠标确认选择，区域坐标将显示在界面上
   - 选择后会自动显示区域预览图
   - 按 ESC 键可取消选择

3. **配置参数**
   - **空闲阈值** - 画面静止多少秒后触发警报（默认 30 秒）
   - **自定义MP3** - 点击"浏览"选择本地MP3文件
   - **MP3播放次数** - 触发警报时MP3播放几次（默认 1 次）
   - **显示器编号** - 选择在哪个显示器上进行区域选择（默认 1）

4. **测试MP3** - 点击"测试MP3"按钮可预览MP3效果

5. **开始监控** - 点击"开始监控"按钮，状态变为绿色表示正在监控
   - 开始监控后，"选择区域"按钮会被禁用

6. **停止监控** - 点击"停止监控"按钮可随时停止
   - 停止监控后，"选择区域"按钮恢复可用

## 工作原理

1. 程序使用 `mss` 库对指定区域进行屏幕截图（每 1 秒一次）
2. 将截图转换为 NumPy 数组提取 RGB 像素值
3. 与上一张截图进行逐像素比较，计算平均差异
4. 如果平均像素差异超过阈值（默认 5%），认为画面有活动
5. 如果连续检测到画面无变化超过设定阈值，触发MP3报警（或蜂鸣）
6. 报警后，只有当画面恢复活动后才会再次触发报警

## 状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|
| 待机 | 灰色 | 未开始监控 |
| 监控中 (有活动) | 绿色 | 检测到画面有变化 |
| 画面静止 X 秒 | 橙色 | 检测到画面静止，正在计时 |
| 警报: 画面静止 X 秒! | 红色 | 超过阈值，触发报警 |

## 常见问题

### 程序启动失败/闪退

- 确保已安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
- 运行 `启动监控.bat` 自动安装依赖
- 查看同目录下的 `vibe_sentinel_error.log` 日志文件

### MP3文件不播放

- 确保选择的是有效的MP3文件
- 检查系统音量设置
- 确认不是在静音状态

### 区域选择后预览不显示

- 确保选择的区域大小超过 10x10 像素
- 检查日志文件中的错误信息

### 多显示器如何选择？

- 在"显示器编号"中设置要选择的显示器序号
- 显示器编号从 1 开始，对应 `mss` 库中的 monitors[1]

## 技术栈

- **屏幕捕获**: [mss](https://github.com/BoboTiG/python-mss)
- **图像处理**: [Pillow](https://github.com/python-pillow/Pillow) + [NumPy](https://numpy.org/)
- **图形界面**: Tkinter (Python 内置)
- **MP3播放**: [playsound](https://pypi.org/project/playsound/)
- **打包工具**: PyInstaller
